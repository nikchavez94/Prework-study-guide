HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag contains information about how the browser receives the webpage

CSS

  • CSS allows the webpage to have more style
  • This may include colors and font styles
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a platform that allows coders to work on projects together
  • It works by using branches that later will be integrated together
  • There are specific commands to push and pull the latest code from Git to your Terminal
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • To add a new branch you must start from the main branch
  • Then you will need to "push" all the changes to git so they can be integrated
